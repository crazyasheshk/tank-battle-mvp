## 📦 本地运行

如果你想在本地运行或修改游戏：

1. 克隆仓库
```bash
git clone https://github.com/crazyasheshk/tank-battle-mvp.git
cd tank-battle-mvp
```

2. 使用 Godot 4.6+ 打开项目
```bash
godot project.godot
```

3. 或者运行本地 Web 服务器
```bash
./start-web-server.sh
# 访问 http://localhost:8890
```
