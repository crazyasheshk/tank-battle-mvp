## 一、游戏概述

### 1.1 游戏概念
- **类型**: 2D 俯视角坦克射击游戏
- **风格**: 极简风格，快速原型验证
- **核心体验**: 移动 → 射击 → 击毁敌人 → 胜利

### 1.2 目标用户
- 喜欢射击类游戏的休闲玩家
- 寻找快节奏对战体验的玩家

### 1.3 开发目标
- 验证坦克对战核心玩法
- 测试 AI 敌人行为逻辑
- 为后续扩展奠定架构基础

## 二、游戏玩法

### 2.1 核心循环

```
开始游戏 → 移动探索 → 发现敌人 → 战斗 → 击毁目标 → 胜利/失败 → 重新开始
```

### 2.2 操作方式

| 按键 | 功能 |
|-----|------|
| W | 向上移动 |
| S | 向下移动 |
| A | 向左移动 |
| D | 向右移动 |
| 空格 | 发射子弹 |
| F5 | 开始/继续游戏 |
| F6 | 暂停/继续 |
| F7 | 重新开始 |

### 2.3 游戏目标

**胜利条件**: 击毁所有敌人 + 击毁红色目标

**失败条件**: 玩家血量归零

## 三、角色设计

### 3.1 玩家坦克

| 属性 | 数值 |
|-----|------|
| 外观 | 绿色方块 + 灰色炮管 |
| 血量 | 5 HP |
| 移动速度 | 200 像素/秒 |
| 射击冷却 | 0.3 秒 |
| 子弹伤害 | 1 |

**特性**:
- 自由四向移动
- 坦克朝向移动方向
- 血量条显示（头顶 + UI）
- 受击闪烁反馈

### 3.2 敌人坦克（AI）

| 属性 | 数值 |
|-----|------|
| 外观 | 紫色方块 + 灰色炮管 |
| 血量 | 3 HP |
| 巡逻速度 | 40 像素/秒 |
| 追击速度 | 80 像素/秒 |
| 射击频率 | 1.5 秒/发 |
| 检测范围 | 300 像素 |
| 攻击范围 | 200 像素 |

**AI 行为状态**:

| 状态 | 触发条件 | 行为 |
|-----|---------|------|
| 巡逻 | 默认/玩家离开检测范围 | 随机方向移动，定时改变方向 |
| 追击 | 玩家进入检测范围 | 向玩家移动 |
| 攻击 | 玩家进入攻击范围 | 停止移动，瞄准射击 |

### 3.3 目标物体

| 属性 | 数值 |
|-----|------|
| 外观 | 红色方块 |
| 血量 | 3 HP |
| 分数 | +100 |

**特性**:
- 静止不动
- 击毁后触发胜利
- 受击闪烁反馈

## 四、战斗系统

### 4.1 子弹属性

| 类型 | 玩家子弹 | 敌人子弹 |
|-----|---------|---------|
| 外观 | 橙色方块 | 红色方块 |
| 飞行速度 | 400 像素/秒 | 300 像素/秒 |
| 最大距离 | 1000 像素 | 800 像素 |
| 伤害 | 1 | 1 |

### 4.2 伤害计算

- 直线伤害：每次命中固定伤害值
- 无护甲/减伤机制
- 血量归零即死亡

### 4.3 碰撞系统

| 对象类型 | Layer | Mask |
|---------|-------|------|
| 玩家 | 1 (player) | 7 (all) |
| 敌人/目标 | 2 (enemy) | 1 (player) |
| 玩家子弹 | 4 (projectile) | 7 (all) |
| 敌人子弹 | 4 (projectile) | 3 (player+enemy) |

## 七、技术架构

### 7.1 项目结构

```
tank-battle-mvp/
├── project.godot           # 项目配置
├── scenes/                 # 场景文件
│   ├── Main.tscn          # 主场景
│   ├── Player.tscn        # 玩家
│   ├── Enemy.tscn         # 敌人
│   ├── Bullet.tscn        # 玩家子弹
│   ├── EnemyBullet.tscn   # 敌人子弹
│   ├── Target.tscn        # 目标
│   ├── GameUI.tscn        # 游戏UI
│   └── VictoryEffect.tscn # 胜利特效
├── scripts/               # 脚本文件
│   ├── Player.gd          # 玩家控制
│   ├── Enemy.gd           # 敌人AI
│   ├── Bullet.gd          # 玩家子弹
│   ├── EnemyBullet.gd     # 敌人子弹
│   ├── Target.gd          # 目标逻辑
│   ├── GameManager.gd     # 游戏管理器
│   ├── GameUI.gd          # UI控制器
│   ├── VictoryEffect.gd   # 胜利特效
│   └── Main.gd            # 主场景控制
└── README.md              # 项目说明
```

### 7.2 核心模块

| 模块 | 职责 |
|-----|------|
| GameManager | 游戏状态管理、分数、暂停、快捷键 |
| Player | 玩家移动、射击、血量 |
| Enemy | AI行为（巡逻/追击/攻击） |
| Bullet | 子弹飞行、碰撞检测 |
| GameUI | 界面显示、状态同步 |

### 7.3 设计模式

- **单例模式**: GameManager (Autoload)
- **状态模式**: Enemy AI 状态机
- **观察者模式**: 信号系统解耦模块
- **对象池**: 子弹复用（可扩展）
